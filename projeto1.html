Descrição

Sistema web para compra de produtos e controle de estoque, com área administrativa e integração com banco de dados MySQL. Desenvolvido em PHP, HTML e CSS.

Funcionalidades

  • Visualização de produtos disponíveis
  • Reserva/compra de produtos
  • Cadastro de novos produtos
  • Controle de estoque
  • Área administrativa com restrição de acesso

Objetivo do projeto

Aplicar conceitos de desenvolvimento web e integração de banco de dados, simulando um sistema real de compra e gestão de estoque no ambiente escolar.
